나의 강점 발견 진단 (Strengths Finder 약식 도구)

갤럽(Gallup)의 CliftonStrengths 34가지 테마를 기반으로 설계된 약식 자가 진단 웹 도구입니다. 개인적인 강점 발견과 성장을 돕기 위한 목적으로 제작되었습니다.

🚀 주요 기능

68문항 진단: 34가지 테마별 2개 문항을 통해 정밀도 향상

자동 넘김 인터페이스: 답변 클릭 시 즉시 다음 문항으로 이동하여 빠른 진단 가능

상위 5대 강점 분석: 진단 결과에 따른 TOP 5 강점과 활용 팁 제공

영역별 분포 차트: 실행력, 영향력, 관계 형성, 전략적 사고 4대 영역 점수 시각화

이전 결과 저장: 브라우저 localStorage를 활용하여 새로고침 시에도 결과 유지

사용자 분석 통합: Microsoft Clarity를 통한 익명 행동 데이터 분석

🛠 사용 기술

HTML5 / CSS3 (커스텀 변수 및 애니메이션)

Vanilla JavaScript (Fisher-Yates 셔플, 점수 산출 로직)

FormSubmit (백엔드 없는 피드백 전송)

Microsoft Clarity (사용자 경험 분석)

⚠️ 면책 조항 (Disclaimer)

본 도구는 공식 CliftonStrengths 진단이 아닙니다.

갤럽(Gallup)의 34가지 테마 정의를 기반으로 한 참고용 약식 도구입니다.

더 정확하고 공식적인 결과 보고서는 store.gallup.com에서 유료 진단을 통해 확인하시기를 권장합니다.
